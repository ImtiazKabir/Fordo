About Fordo

  • 📁 Open-source: Fordo is an open-source todo app built around simplicity.
  • 💻 Written in ANSI C: The entire backend is built in ANSI C, utilizing SQLite3 and 'imlib', a custom OOP library by the author.
  • ⚙️ Flexible Server: The project demonstrates that ANSI C can still be powerful, with a system call-based C server that mimics Express.js functionality.
  • 🔧 Pedantic Design: Built with abstractions, inheritance, and error handling—all in pedantic ANSI C.
  • 🎨 Beautiful Frontend: The frontend uses vanilla JavaScript and CSS to complement the clean design.
  • 🔗 GitHub: Check out the project on GitHub

About Imlib

  • 🔧 Custom OOP Library: Imlib was created to showcase the power of ANSI C with full support for OOP features.
  • 📐 Object-Oriented Power: Imlib recreates all the fundamental pillars of OOP with enhanced flexibility.
  • 🔄 Design Patterns: Easily implements design patterns, making complex coding more manageable.
  • 💡 Modern Features: Features like runtime polymorphism, error handling, and thread-safe logging enhance the coding experience.
  • 🛠️ Standard Containers: Includes implementations for containers like LinkedLists, HashMaps, and StringBuilders with inheritance support.
  • 📚 GitHub: Explore Imlib on GitHub
